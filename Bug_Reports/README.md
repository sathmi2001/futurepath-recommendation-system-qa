# Bug Reports

## Overview

This folder contains the defects identified during the manual testing of the **Future Path Recommendation System**.

Each reported bug was documented after executing the planned test cases and includes sufficient information for developers to reproduce and resolve the issue.

---

## Bug Report Details

Each bug report includes:

- Bug ID
- Module
- Bug Summary
- Description
- Steps to Reproduce
- Expected Result
- Actual Result
- Severity
- Priority
- Status
- Environment
- Reporter

---

## Severity Levels

- Critical
- High
- Medium
- Low

---

## Priority Levels

- High
- Medium
- Low

---

## Objective

The purpose of these bug reports is to clearly communicate software defects, assist developers in reproducing issues, and support the defect resolution process while improving the overall quality of the application.
