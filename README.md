# Future Path Recommendation System - Manual Testing

## Project Overview

This repository contains the manual testing artifacts created for the **Future Path Recommendation System**, a web application designed to help school leavers identify suitable career paths based on their skills and experience.

The testing activities were carried out based on the provided Software Requirement Specification (SRS) and focused on validating the core functionalities of the application through structured test cases and defect reporting.

---

## System Description

The Future Path Recommendation System is a web-based platform that enables users to:

- Register and log in to the system
- Create and manage their profile
- Upload resumes
- Enter skills and experience
- Receive personalized career recommendations
- Access learning resources
- Search for jobs and gigs
- Communicate through the messaging system

---

## Testing Scope

Manual testing was performed on the major functional modules described in the SRS, including:
